# Agent-Generated Documentation

This directory contains documentation and analysis generated by AI agents during development.

**Purpose:** Summary documents, architectural notes, and implementation explanations created during feature development and bug fixes.

**Guidelines:**
- Document significant changes, design decisions, and investigation results
- Name clearly: `{feature}_{OUTCOME}.md` or `{topic}_SUMMARY.md`
- Examples:
  - `TOUCHING_SUBPLOTS.md` - Implementation of perfectly-touching subplots
  - `AXES_TOUCHING.md` - Investigation of axis touching behavior
  - `CONFIG_SYSTEM_SUMMARY.md` - Config system design overview
  - `SHARED_AXES_IMPROVEMENT.md` - Enhancement documentation
  - `MARKER_SIZE_FIX.md` - Bug fix analysis
  - `test_graphics_output_README.md` - Notes on test graphics generation

These are reference documents for understanding implementations, not user-facing documentation.
