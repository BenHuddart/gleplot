# Agent-Generated Tests

This directory contains test scripts generated by AI agents during development and exploration.

**Purpose:** Temporary or exploratory tests that validate specific features during development but are not part of the main test suite.

**Guidelines:**
- Scripts here are optional test runs, not core validation
- Use clear names: `{feature}_test.py` or `{exploration}_test.py`
- Document what the test validates at the top of the file
- These tests should be runnable from the repo root: `python tests/agent/{script_name}.py`
- Do NOT rely on automatic pytest discovery (not named `test_*.py`)
- Output from these tests should NOT be committed (add to `.gitignore`)

**Examples:**
- `custom_prefix_test.py` - Validates custom data file prefix feature
- Feature-specific exploratory tests during development
