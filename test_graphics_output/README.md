## Inspection Notes

### Shared Axes Validation

The shared axes feature (tests 8-10) should show:

✅ **Proper spacing reduction**:
- Shared x-axis: vertical spacing ~0.8cm (instead of 2.0cm)
- Shared y-axis: horizontal spacing ~0.5cm (instead of 1.5cm)

✅ **Correct label/tick visibility**:
- Check GLE files for `xaxis ... off` and `yaxis ... off` commands
- Only appropriate edges should show labels

✅ **Synchronized limits**:
- All subplots with shared axes should have identical axis ranges
- Look for matching `min` and `max` values in GLE files

✅ **No overlapping elements**:
- Plot areas should not overlap
- Labels should not collide
- Graphs should be properly positioned

## How to Inspect

### View PDF files
```bash
open test_graphics_output/test_*.pdf
```

### View GLE source
```bash
cat test_graphics_output/test_08_shared_x_axis.gle
```

### Compare shared vs non-shared
Compare:
- `test_07_subplots_basic.gle` (no sharing) 
- `test_08_shared_x_axis.gle` (x-axis shared)
- `test_10_shared_both.gle` (both shared)

Look for differences in:
- `amove` positioning coordinates
- `size` commands
- `xaxis`/`yaxis` visibility (`off` keyword)
- Spacing between graph blocks

### Recompile GLE files
```bash
# Single file
gle test_graphics_output/test_08_shared_x_axis.gle -d pdf

# All files
for f in test_graphics_output/*.gle; do gle "$f" -d pdf; done

# Different format
gle test_graphics_output/test_08_shared_x_axis.gle -d png -r 300
```

## Notes for Developers

The shared axes implementation involved:

1. **Figure class** (`figure.py`):
   - Added `sharex` and `sharey` parameters
   - Automatic spacing adjustment
   - Axis limit synchronization

2. **Axes class** (`axes.py`):
   - Visibility control flags (`_show_xlabel`, `_show_ylabel`, etc.)

3. **Writer class** (`writer.py`):
   - Support for conditional axis display using GLE `off` command

4. **API** (`__init__.py`):
   - Updated `subplots()` function signature

The implementation maintains backward compatibility while adding powerful new layout control.
